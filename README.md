# 真家GAIAホームページ

Gift Link Maie / 真家GAIA コミュニティの公式ランディングページです。

## 構成

- `index.html` — メインページ（利用規約・プライバシーポリシー・申込フォーム含む）
- `css/style.css` — スタイル
- `js/main.js` — ルーティング・FAQ・フォーム等
- `js/legal-content.js` — 利用規約・プライバシーポリシー本文
- `assets/images/` — 画像アセット

## ローカルでの確認

ブラウザで `index.html` を開くか、簡易サーバーで表示してください。

```bash
# Python がある場合
python -m http.server 8080
```
